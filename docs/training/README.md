# Training Deck

This folder is reserved for the company-template training deck.

Recommended output file:

```text
MC_QR_Manager_User_Training_v2.1.0.pptx
```

The training deck should be a short operator education deck, not the source manual.

- Source manual: `docs/user-guide.html`
- Field distribution: `docs/MC_QR_Manager_User_Guide_v2.1.0.pdf`
- Training deck: `docs/training/MC_QR_Manager_User_Training_v2.1.0.pptx`

Use the company PowerPoint template and keep each slide focused on one operational message.
