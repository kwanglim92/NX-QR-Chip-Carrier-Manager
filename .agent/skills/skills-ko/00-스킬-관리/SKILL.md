# SKILL 00 — 스킬 관리 & 레지스트리

## 개요
모든 스킬 문서의 생성, 수정, 검색, 의존성 추적을 관리하는 메타 스킬입니다.

## 핵심 구조
- **`registry.json`**: 전체 스킬 메타데이터 (ID, 이름, 태그, 의존성, 소스 파일)
- **이중 언어 폴더**: `skills-en/` (실무) + `skills-ko/` (참고용)
- **신규 추가 절차**: ID 할당 → SKILL.md 생성 → registry 업데이트 → 검증

## 버전 규칙
| 변경 | 버전 업 |
|------|---------|
| 오탈자 | 없음 |
| 코드 예제 추가 | Minor (x.Y) |
| 패턴/API 추가 | Major (X.0) |

## 주의사항
- 스킬 ID는 절대 삭제/재사용 금지
- 한국어 버전은 참고용 요약본 (영어 판 완전 번역 아님)
